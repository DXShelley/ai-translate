# 更新日志

- 将当前工作区改动整理为 v3.0.0 发布版本。

## v2.0.1

### 功能优化

1. **并行请求优化**
   - 划词翻译英文单词时，翻译请求和词典查询并行发送
   - 减少用户等待时间，约节省一半时间

2. **自定义模型选项**
   - 已部署模型下拉框新增"自定义模型..."选项
   - 选择后可手动输入模型名称并保存
   - 不会在获取模型列表后被清除

3. **删除模型保护**
   - 删除模型时检查启用的模型数量
   - 如果只剩一个启用的模型，弹框提示"至少需要保留一个启用的模型配置"

4. **请求日志优化**
   - 日志保留条数从 50 条减少到 15 条
   - 清空日志时弹框确认

### 配置页布局

调整后的顺序：
1. 接口预设
2. 模型接口（包含温度、超时，已部署模型含自定义选项）
3. 翻译参数
4. 前端交互
5. 提示词
6. 模型配置文本
